# Requirements Analysis

## Project Name

HireWise AI

## Objective

To build a full-stack AI-powered recruitment management system where candidates can apply for jobs and admins can manage jobs, applications, resume screening, interviews, and automated email notifications.

## User Roles

### Admin / HR

- Login
- Create jobs
- View applications
- Shortlist or reject candidates
- Schedule interviews
- View dashboard analytics

### Candidate

- Register
- Login
- View jobs
- Apply for jobs
- Upload resume
- Track application status

## Completed Requirements 30.5.2026

- User registration
- User login
- Password hashing
- JWT authentication
- Protected route access

## Upcoming Requirements

- Job posting system
- Application system
- Resume upload
- AI-based resume scoring
- Email automation
- Interview scheduling
- Dashboard analytics
