# HireWise AI

HireWise AI is an AI-powered Recruitment Management System built using the MERN stack.

The platform allows candidates to apply for jobs and allows admins/HR to manage jobs, applications, interviews, resume screening, and automated communication.

## Current Progress 30.5.2026

- Backend server setup completed
- MongoDB Atlas connected
- User registration API completed
- User login API completed
- Password hashing implemented using bcrypt
- JWT token generation implemented
- Authentication middleware created
- Protected profile route tested successfully
